# Haitiart - Agencia Inmobiliaria

Sitio web para Haitiart, agencia especializada en la venta de planos inmobiliarios en República Dominicana.

## 🚀 Características

- **Diseño Responsivo**: Optimizado para desktop, tablet y móvil
- **Accesibilidad**: Cumple con estándares WCAG con atributos ARIA
- **Modal Interactivo**: Sección "Quién Somos" con modal accesible
- **Múltiples Opciones de Pago**: Visa, Mastercard, Efectivo, Transferencia y Crypto
- **Animaciones Suaves**: Transiciones y efectos hover optimizados
- **Código Optimizado**: CSS y JS ligeros para mejor rendimiento

## 🏗️ Estructura del Proyecto

```
haitiart/
├── haitiart.html          # Página principal
├── assets/
│   ├── agencyStyle/
│   │   └── haitiart.css   # Estilos optimizados
│   └── JS/
│       └── haitiart.js    # JavaScript funcional
├── data/

└── .gitignore            # Configuración Git
```

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Diseño responsivo con Flexbox/Grid
- **JavaScript**: Interactividad y accesibilidad
- **JSON**: Almacenamiento de datos

## 🚀 Instalación y Uso

### Opción 1: Clonar desde GitHub
```bash
git clone https://github.com/TU_USUARIO/haitiart.git
cd haitiart
```

### Opción 2: Descargar ZIP
1. Ve a https://github.com/TU_USUARIO/haitiart
2. Click en "Code" > "Download ZIP"
3. Extrae el archivo y abre en tu editor

### Ejecutar localmente
```bash
# Usando Python (recomendado)
python -m http.server 8000

# O usando Node.js
npx serve .

# O simplemente abre haitiart.html en tu navegador
```
